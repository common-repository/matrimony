== Description ==

**Matrimony** is a Matrimonial plugin which converts your WordPress website to a Matrimonial Website. Candiates can register their profiles and can see other registered profiles. Make a wedding profile website using this plugin.

= Matrimony Shortcodes =

To show Matrimonial form
> `[mwa_matrimonial]`

To show profiles in page or post
> `[mwa_matrimonial_page]`

To show profiles in slider
> `[mwa_matrimonial_slider]`

### Matrimony Features

* Save the candidates profile & details
* Show profiles in the slider with setting of male or female
* Admin has control to show and hide the profile capabilities
* Multi-language support

== Installation ==

**Matrimony [Installation Guide]**

1. You can:
 * Upload the entire `matrimony` folder to the `/wp-content/plugins/` directory via FTP.
 * Upload the zip file of plugin via *Plugins -> Add New -> Upload* in your WordPress Admin Panel.
 * Search **Matrimony** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate the plugin through *Plugins* menu in WordPress Admin Panel.
3. Click on *MWA Matrimonial* in menu.
4. Ready, now you can use it using shortcode on page or post.
